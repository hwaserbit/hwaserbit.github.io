# hwaserbit Log 🚀

[![Gem Version](https://img.shields.io/gem/v/jekyll-theme-chirpy)][gem]&nbsp;

**친절한 서버 엔지니어, hwaserbit의 기술 블로그 저장소입니다.**
이 블로그는 [Jekyll](https://jekyllrb.com/)과 [Chirpy][chirpy] 테마를 사용하여 구축되었습니다.

> **🌐 Live Blog:** [https://hwaserbit.github.io](https://hwaserbit.github.io)

## 📝 소개 (Introduction)

서버 인프라, 백엔드 개발, 그리고 개발 과정에서 마주친 트러블 슈팅 경험을 기록합니다.
단순한 코드 저장이 아닌, **"문제 해결의 과정과 고민"**을 담는 공간을 지향합니다.

## 🛠️ 기술 스택 (Tech Stack)

* **Generator:** Jekyll (Static Site Generator)
* **Theme:** [Chirpy Starter][chirpy-starter]
* **Hosting:** GitHub Pages & Actions
* **Comment:** Giscus (예정)

## 📂 디렉토리 구조 (Structure)

이 저장소의 주요 파일 구조는 다음과 같습니다.

```shell
.
├── _config.yml    # 사이트 환경 설정 (Title, URL, Author 등)
├── _posts         # 작성된 블로그 포스트들 (Markdown)
├── _tabs          # 상단 메뉴 페이지 (About, Categories 등)
└── assets         # 이미지(로고, 배너) 및 정적 파일


시간을 절약하고 복사하는 동안 일부 파일이 손실되는 경우를 대비하여 해당 파일/구성을 추출합니다.
최신 버전의 **Chirpy** 테마와 [CD][CD] 작업 흐름을 여기로 보내면 몇 분 안에 글쓰기를 시작할 수 있습니다.

## 기여

이 저장소는 테마 저장소의 새 릴리스로 자동 업데이트됩니다. 문제가 발생하거나 개선에 기여하고 싶다면 [테마 저장소][chirpy]를 방문하여 피드백을 제공해 주세요.
